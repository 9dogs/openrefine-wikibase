Wikidata reconciliation for OpenRefine

This web service can be used to align datasets to Wikidata items in OpenRefine.

Use the following URL in OpenRefine: https://tools.wmflabs.org/openrefine-wikidata/en/api.

Replacing "en" by another language code will display items and properties in your language, when they are available.

This interface works with OpenRefine 2.6 rc2. It is not compatible with Google Refine 2.5.

Configure the reconciliation

You can choose a type to reconcile rows against. Only items which are instances of this type (or any subclass of it) will be returned. It is possible that the types suggested by the interface are too specific for your dataset as the algorithm only uses the first rows to guess the type. Do not hesitate to choose a more general type (for instance by choosing human settlement (Q486972) instead of city (Q515)).

You can match columns in your dataset to Wikidata properties. This will help refine the matches, for better precision.
